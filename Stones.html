Stone Tool Assembly using AR

In order to simplify the process of assembling stone tools, this AR application can be used to detect the objects and overlay different graphics and display the order and method of joining all stone tool pieces.

石器を組み立てるプロセスを簡略化するために、このARアプリケーションを使用して、オブジェクトを検出し、さまざまなグラフィックをオーバーレイし、すべての石器のピースを結合する順序と方法を表示できます。

Used Software:
Unity 3D, Autodesk Maya, Vuforia,Photoshop, ARCore
