Crow Attack!

Crow Attack! is a Kinect game where you pose and move in various positions to control the main character. Players can gain points by attacking people within a set period of time. Each character has a different difficulty level, so finding and choosing your opponents carefully is an important gameplay element.

Role:
Modelling, rigging and animation of all environment and character assets (Autodesk Maya, Photoshop, Unity 3D)
Programming UI elements and gameplay design (Unity 3D - C#).
